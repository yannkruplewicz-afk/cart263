# RoboShop

Ray Hernaez (40268579), Yann Kruplewicz (40339621), Zhuoheng Yang (40311134)

[Project Link](https://sleepyrayray.github.io/cart263/projects/project-2/)

## Description

RoboShop is an interactive web experience where you visit a futuristic robot shop, answer a short questionnaire, record a short greeting, and reveal the robot that fits you best. The project is designed as a playful experience with a simple design and interactions.

## How To Play

1. Open the `Guide` on the main menu
2. Agree to the voice recording consent
3. Answer the robot matching questions
4. Record a short greeting for your robot
5. Click the package to reveal your robot
6. Purchase your robot and return to the menu

## Main Features

- 5 robot result types
- 5 robot colour choices
- 25 total robot image combinations
- voice recording and robot voice playback
- animated package reveal
- short robot description and purchase ending

## Tools Used

- `p5.js`
- `Three.js`
- local JSON data with `fetch()`
